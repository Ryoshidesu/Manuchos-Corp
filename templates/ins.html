Bienvenido a la seccion de la instalacion de Flask

Pasos a seguir

  • Instalar python la version mas reciente junto con Visual studio code

  • Iniciamos una nueva terminal y escribe el siguente comando.
  • El comando es el siguiente: pip install Flask.
